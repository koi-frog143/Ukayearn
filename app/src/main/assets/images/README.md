Asset drop folders for Ukayearn.

Place your image files in:

- `profiles/` for user and seller profile photos
- `stores/` for default store images
- `items/` for product/item images
- `categories/` for category thumbnails or icons

These folders are under Android assets so they can be organized freely. If you want to reference images directly from XML later, copy selected final images into `app/src/main/res/drawable` with lowercase snake_case names.
